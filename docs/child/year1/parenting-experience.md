# 欢迎你的到来，豌豆🫛
昨天晚上突然兴奋的睡不着，我明白这是我的感觉总是后知后觉；从你初生到现在我好像直到昨晚才真正感受到那种震惊的感觉，你是我的孩子——一个活脱脱的新人类，我才刚习惯你还在妈妈肚子里，你就这么正式和我们约见面了！我从此刻开始真正感受到了生命不一样的那部分。好像感觉自己有了用另一种方式存在的意义。

“齿轮开始新一轮的转动，我仿佛听到了上帝在翻页的声音...”

（稍等会，你妈妈在和我聊天，我先出戏一会...）
